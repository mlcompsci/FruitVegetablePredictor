# Data Directory

## Structure

```
data/
  raw/                  # Place your dataset here
    apple/
      img001.jpg
      img002.jpg
      ...
    banana/
      ...
    carrot/
      ...
  processed/            # Generated by prepare_data.py
    train/
      apple/
      banana/
      ...
    val/
      ...
    test/
      ...
```

## Instructions

1. Download or collect images of fruits and vegetables.
2. Organize them into `data/raw/<class_name>/` folders.
3. Each folder name becomes the class label.
4. Supported formats: `.jpg`, `.jpeg`, `.png`, `.bmp`, `.webp`.
5. Run `python -m scripts.prepare_data` to create stratified splits.

## Recommended Datasets

- [Fruits 360](https://www.kaggle.com/moltean/fruits) on Kaggle
- [Vegetable Image Dataset](https://www.kaggle.com/datasets/misrakahmed/vegetable-image-dataset) on Kaggle
- Custom photos taken with your phone
